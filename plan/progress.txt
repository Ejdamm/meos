## Codebase Patterns
- Every `#include "..."` directive MUST match the exact filename casing on disk (case-sensitive for Linux).
- Use `python3 verify_includes.py` to check for casing issues.
- Use global string utility functions from `meos_util.h` for conversions (widen, narrow, toUTF8, fromUTF8, recodeToWide, recodeToNarrow).
- Use standard C++ types (uint32_t, uint8_t, bool, etc.) in domain files and include <cstdint> in headers.
- Be careful with BOOL in SQL strings; it might be a database-specific type (like in MySQL/SQLite) that should remain as is.
